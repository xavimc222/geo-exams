# Geography Quiz Application

A React-based geography quiz application covering Poland, Germany, and Austria.

## Features

- Interactive geography quizzes with multiple choice questions
- Support for multiple languages (English/Czech)
- Filtering options by country and question type
- Real-time scoring system
- Responsive design that utilizes full horizontal space

## Getting Started

1. Navigate to the quiz-app directory:
   ```bash
   cd quiz-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## Project Structure

- `/quiz-app/` - Main React application
- `/data/` - XML data files with geography information
- `/_prompts/` - Development prompts and planning documents

## Technologies Used

- React
- Vite
- CSS3
- JavaScript ES6+
